# DAVIS — одностраничный сайт

Статический лендинг (HTML/CSS/JS) + обработчик формы `send.php`.

## Структура

- `index.html` — страница
- `assets/` — стили, скрипты, изображения
- `send.php` — обработчик формы

## Локальный запуск

Открой `index.html` в браузере или используй любой локальный сервер (например, расширение Live Server).
